Admin management application for EveryDukan platform built with Next.js, enabling efficient platform operations management.

## Features

- **Admin Management**: User, order, and inventory management tools
- **Real-Time Updates**: Seamless backend service integration
- **Modern UI**: Fast, server-side rendering with smooth interactions
